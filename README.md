# *BarberShop Software*
_**Software de agendamento para Barbearia**_
---
Desenvolvi um software para uma barbearia juntamente com 3 amigos para o TCC da ETB(Escola Técnica de Brasília);\
CRUD de Clientes e usuários;\
Tecnologias usadas:
* `Bootstrap`
* `SERVLET`
* `LOMBOK`
* `MySql`
* `JSTL`
* `Java`
* `HTML`
* `CSS`
* `JSP`
